Topik 31: Persyaratan dan format database karyawan untuk bulk upload (kolom A–F wajib; first name + last name; email unik; employee code unik; bank code & account jika auto-disbursement)
Total informasi yang ditemukan pada topik ini: 10 poin

1. Semua karyawan yang akan mengakses Mekari Expense harus diberikan akses melalui database employee. Mekari Expense
2. Template bulk upload memiliki kolom A sampai F yang bersifat wajib.
3. Employee name harus terbagi menjadi first name dan last name (dua bagian).
4. Jika perusahaan hanya memiliki satu nama karyawan, boleh mengulangi nama pada field first/last name.
5. Employee code disesuaikan dengan employee code di perusahaan.
6. Phone number wajib diisi.
7. Email karyawan digunakan sebagai akses login; email harus benar dan unik.
8. Employee code dan employee email bersifat unik dan tidak dapat diubah kemudian hari; jika terjadi salah input harus invite ulang dengan kombinasi berbeda.
9. Jika perusahaan menggunakan auto-disbursement, perlu mengisi bank code dan bank account number di database.
10. Setelah upload, sistem akan mengirimkan email invitation dari Mekari ke karyawan untuk membuat akun/password.
